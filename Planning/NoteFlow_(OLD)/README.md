# Notes / Flash Cards Website (StudySync / NoteFlow)

## Description
Student focused website that allows student to take notes, create flashcards, share notes and flashcards, and with a built in AI-powered quiz generator into a single workspace

---

## Features

### Notes
- Write notes in Markdown with live preview (with icons at the top so students dont have to remember how to format it)
- Edit, delete and organise notes with tags for easy filtering
- Copy any shared notes into your own account to personalize it
- Generate an AI quiz directly from any note

### Flashcards
- Create decks with a title and tags, independent from notes
- Add cards with a front and back, and allow them to flip
- Mark cards right or wrong, at the end you can see which you got right and which you got wrong (doesnt save)
- Copy any shared flashcards into your own account
- (Maybe) AI generate flashcards from note?

### AI Quiz
- Generate a quiz from any notes. The AI reads the content and makes questions and mcq automatically
- Choose to save quiz / discard / delete
- See quiz history, see questions, your answers, and if it is correct
- Retake a previous quiz, or start fresh with a new one

### Future Features
- Allow collaboration on a single doc or flashcard (requires hosting)
- Allow AI quiz to have written quiz answers (checked by AI) can be paid feature
- Make and share your own quiz
- Folders to better organise notes and cards
- Friend Request

## User Stories

### Authentication
- As a student, I want to register an account with a username, email and password so my content is saved to me
- As a student, I want to log in and out so my content stays private

### Notes
- As a student, I want to create notes using a rich text editor with a formatting toolbar so I don't have to remember Markdown syntax
- As a student, I want to edit and delete my notes
- As a student, I want to tag my notes so I can filter and find them by topic
- As a student, I want to mark a note as public so other students can discover and copy it
- As a student, I want to copy a public note into my own account to personalise it
- As a student, I want to generate an AI quiz from any of my notes

### Flashcards
- As a student, I want to create a flashcard deck with a title and tags
- As a student, I want to add cards to a deck each with a front and back
- As a student, I want to flip through cards in a study session
- As a student, I want to mark each card as right or wrong during a session
- As a student, I want to see a results summary at the end of a session showing which cards I got right and wrong
- As a student, I want session results to not be saved so there is no pressure and it is just for self checking
- As a student, I want to mark a deck as public so other students can discover and copy it
- As a student, I want to copy a public deck into my own account to personalise it

### AI Quiz
- As a student, I want to be informed that the quiz is generated by AI and therefore would not 100% be accurate
- As a student, I want to generate an MCQ quiz from a note so the AI formulates the questions and options for me
- As a student, I want to select answers from multiple choice options
- As a student, I want to choose to save or discard a quiz after completing it
- As a student, I want to view my saved quiz history for any note
- As a student, I want to see each question, the options, my answer, and whether it was correct in the history view
- As a student, I want to delete a saved quiz from my history
- As a student, I want to retake a saved quiz with the same questions
- As a student, I want to generate a fresh quiz on the same note for new questions
- As a student, I want my quiz history to be private

### Discovery
- As a student, I want to browse public notes and decks from other students
- As a student, I want to filter public content by tag to find material relevant to my units
- As a student, I want to copy any public note or deck into my own account
- As a student, I want to preview a public note before copying it so I can read the full content before adding it to my library
- As a student, I want to preview a public flashcard deck before copying it so I can see all the cards front and back before adding it to my library
